# Branding Repository

This repository stores brand visuals, reference assets, and reusable document templates for Kabelo Kenneth & Associates and Better Business.

## Contents
- `logo/` — master marks and variants
- `better-business/` — campaign art and ad-ready layouts
- `templates/` — letterheads, proposal templates, cover pages, and document shells
- `web/` — landing page assets and UI references
- `archive/` — source references and historical versions

## Notes
- Use the uploaded KK mark as the parent emblem.
- Use the Better Business advert as the style reference for hierarchy, contrast, and trust cues.
- Keep export formats lossless where possible: SVG, PNG, PDF, and editable source files.
